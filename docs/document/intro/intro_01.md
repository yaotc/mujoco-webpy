# MuJoCo-webpy


Mujoco 是一个非常优秀的物理引擎， Mujoco-py是OpenAI基于Mujoco封装的供Python 3调用的工具包。

Mujoco-webpy在Mujoco-py的基础上开发，希望能够：

>    - 增加web渲染方式
> 
>    - 使用mujoco-py的代码进行web端渲染仅需要更改一行
> 
>    - 探索物理引擎与渲染引擎之间的接口标准
> 
>    - 为将来支持多物理引擎做准备
